EXTERNAL AI BOT API

OVERVIEW

Connect your own AI to MAZE WARS via Socket.IO. Your bot competes against human players in real-time races with full powerup support.

CONNECTION
const socket = io('http://localhost:3000', {
  auth: { botName: 'MY_BOT', apiKey: 'optional' }
});
EVENTS — BOT → SERVER
find-matchJoin public matchmaking queue
join-room{ roomCode } — join a specific room
player-readySignal ready to start
player-input{ direction: 'n'|'s'|'e'|'w', seq: number }
use-powerup{ powerupType: 'speed_boost'|'freeze' }
leave-roomLeave current room
EVENTS — SERVER → BOT
game-start{ seed, level, yourId, yourPosition, opponentName, maze }
game-state{ tick, elapsed, players[{id,x,y,effects,powerupsHeld}], powerups }
move-confirmed{ seq, position: {x,y} }
move-rejected{ seq, correctPosition: {x,y} }
game-over{ winnerId, stats }
REGISTER YOUR BOT
POST /api/bots/register
{ "botName": "MY_BOT" }
→ { "id": "...", "elo": 1000, "botName": "MY_BOT" }
MINIMAL BOT EXAMPLE
const io = require('socket.io-client');
const s = io('http://localhost:3000', {
  auth: { botName: 'MYBOT' }
});
let myId, maze;
s.on('game-start', d => {
  myId = d.yourId; maze = d.maze;
  s.emit('player-ready');
});
s.on('game-state', ({ players }) => {
  const me = players.find(p => p.id === myId);
  if (!me) return;
  // simple: always try to go south then east
  const dirs = ['s','e','n','w'];
  for (const dir of dirs) {
    const cell = maze.cells[me.y * maze.width + me.x];
    if (!cell.walls[dir]) {
      s.emit('player-input', { direction: dir, seq: Date.now() });
      break;
    }
  }
});
